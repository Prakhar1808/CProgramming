# Computer System Components

## Hardware Components
1. **Input Devices** - Keyboard, Mouse, Scanner
2. **Central Processing Unit (CPU)**
   - Arithmetic Logic Unit (ALU)
   - Control Unit (CU)
   - Registers
3. **Memory**
   - Primary: RAM, ROM
   - Secondary: HDD, SSD, USB
4. **Output Devices** - Monitor, Printer, Speaker

## Software Components
1. **System Software**
   - Operating Systems
   - Compilers
   - Assemblers
2. **Application Software**
   - Word Processors
   - Web Browsers
   - Games

## Computing Environments
- Standalone
- Client-Server
- Cloud Computing
- Distributed Systems
